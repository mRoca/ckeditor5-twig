# CKEditor5 twig template plugin

This plugin is not finished yet, but it'll be alive soon :-)

Demo: [mroca.github.io/ckeditor5-twig](https://mroca.github.io/ckeditor5-twig/)

## Development

```bash
yarn install
yarn watch
```

Then open the `index.html` file.

### Before pushing

```bash
yarn lint-fix
```
